Tentang Saya

Saya suka menghabiskan waktu luang untuk berselancar di internet. Baik untuk mencari informasi, atau sekedar menikmati konten-konten hiburan yang ada di media sosial.

Saya tertarik untuk mempelajari dan mendalami teknologi digital, khususnya pemrograman karena teknologi informasi kini menjadi tulang punggung global, menciptakan peluang dan tantangan baru bagi masyarakat modern.

Pada pertengahan tahun 2019, saya mulai mencari informasi tentang internet, web, aplikasi mobile, dan sebagainya. Sampai disini saya mulai mengenal dan mencari tahu lebih lanjut bagaimana sistem dan cara kerjanya.

Meski saya tidak memiliki latar belakang tentang pemrograman, namun saya memiliki tekad kuat untuk mulai belajar dan menguasai bidang ini. Di tengah lika-liku hidup yang banyak rintangan, saya sempatkan belajar di waktu luang.

Pengalaman Belajar

Android

Desember 2019 - September 2021

Saya belajar membuat aplikasi android dengan mengikuti setiap panduan dari official web android developer 🡥, dan berbagai sumber lainya untuk bisa lebih memahami.
Selain mempelajari Native API dari android, saya juga mempelajari dari sisi bahasa pemrogramannya baik Java maupun Kotlin, karena tentu saja saya belum memiliki kemampuan pemrograman sebelumnya.

WEB

Maret 2021 - Juli 2021

Untuk web sendiri saya sekedar mengenal saja, belum pernah mendalami atau membuat project pembelajaran. Karena tujuan saya adalah membuat rest api guna mendukung project android yang sedang saya pelajari.
Ini dilatarbelakangi rasa penasaran dan ingin tahu saya setelah banyak mempelajari Firebase, baik yang realtime database maupun cloud firestore serta fitur lainya. Jadi fokus saya mempelajari sistem data dan query, waktu itu saya menggunakan MySQL dan juga framework Laravel.
Pada tahap ini, saya sebenarnya tertarik pada pengembangan aplikasi baik dari sisi Frontend maupun Backend, namun itu semua belum terlaksana karena selanjutnya yang saya tekuni adalah Unity Game Engine.
